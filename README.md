## Features
### 1. 로그인
![001](https://user-images.githubusercontent.com/72987464/154634650-c97b05eb-6e38-46d9-9df6-618e8327d1f8.png)
**1-1. 로그인**
- 직원번호와 비밀번호를 입력하여 로그인할 수 있습니다.
- 5회 이상 틀린 비밀번호를 입력할 경우 계정이 잠깁니다.
- 아이디를 저장할 수 있습니다.


![003](https://user-images.githubusercontent.com/72987464/154634682-49b4c606-08c2-46e9-8cdc-e9bc7204ed68.png)
**1-2. 비밀번호 찾기**
- 아이디, 이름, 주민번호 뒷자리로 비밀번호를 찾을 수 있습니다.
- 인증에 성공하면 비밀번호 변경 페이지로 이동하여 비밀번호를 재설정합니다.


### 2. 대시보드
![007](https://user-images.githubusercontent.com/72987464/154634726-1de53237-9133-4144-9131-b45962b7a3de.png)
**2-1. 점장 직급 대시보드**
- 점장 직급의 계정으로 로그인시 홈화면으로 표시되는 대시보드 화면입니다.
- 점장 외의 직급의 계정 로그인시 대시보드 화면이 아닌 직급에 맞는 화면이 출력되게 됩니다.
- 점장은 자신이 관리하는 **매장의 일/월 매출, 상품별 판매량, 공지사항, 근무 스케줄, 재고상황**에 대해 확인할 수 있습니다.
- JSP에서 쿼리 메소드문으로 받은 결과값을 `HighChart` 라이브러리를 사용해 출력했습니다.


![009](https://user-images.githubusercontent.com/72987464/154634733-5b4e45d9-1309-4229-9e54-2b916d7772ce.png)
**2-2. 지역장 직급 대시보드**
- 지역장 직급의 계정으로 로그인시 홈화면으로 표시되는 대시보드 화면입니다.
- 지역장은 지역내 **지점별 당일 판매 현황, 전지역 매출 현황, 전지점 상품별 월간 판매량**을 확인할 수 있습니다.
- 지역장 직급 이외의 직급은 접근이 제한됩니다.


### 3. 인사관리
![010](https://user-images.githubusercontent.com/72987464/154634760-d27836c1-83ae-47c1-acdb-0cf67a4aed52.png)
**3-1. 직원조회**
- 직원목록을 조회합니다.
- 오른쪽 조직도에서 조직이름을 누르면 해당 조직 내 직원만 조회합니다.
- 오른쪽 조직도에서 아이콘을 누르면 하위 조직이 토글됩니다.


![012](https://user-images.githubusercontent.com/72987464/154634772-5dcb9015-a8f6-40ac-8949-7ce61d2b203d.png)
**3-2. 스태프조회**
- 스태프목록을 조회합니다.
- 매니저/점장/지역장은 최초 진입시 소속 매장/지역을 기준으로 자동으로 선택되어 나옵니다.
- 매니저/점장/지역장은 담당 매장/지역 내 스태프만 조회할 수 있습니다.


![016](https://user-images.githubusercontent.com/72987464/154634811-e2846f78-0ed1-4854-8ca1-2d3f26ffbac9.png)
**3-3. 스태프 근태입력**
- 스태프 번호와 스태프 비밀번호를 입력하여 근태를 입력할 수 있습니다.
- 근태는 현재 시점으로 자동으로 입력됩니다.
- 입력된 근태를 기준으로 일주일의 일 예상 급여를 산출하여 출력합니다.
- 입력 순서가 맞는 버튼만 활성화됩니다.


![018](https://user-images.githubusercontent.com/72987464/154634823-1320ef1d-7d9c-4d7e-9249-602a3b2fbc05.png)
**3-4. 스태프 급여조회**
- 스태프의 월 급여를 조회합니다.
- 스태프 월 급여는 입력된 근태를 기준으로 산출됩니다.
- 현행법에 근거하여 수당과 공제내역을 산출하고 세부산정 내역을 출력합니다.


### 4. 재고관리
![Untitled](https://user-images.githubusercontent.com/72987464/154635883-83a14c5c-c276-432e-8d1e-b22cda1b00a3.png)
**4-1 재고조회**
- 매장에 남아있는 재고 정보를 보여주는 화면입니다.
- 수량 입력 버튼을 눌러 재고 정보 입력 화면으로 이동합니다.


![Untitled (1)](https://user-images.githubusercontent.com/72987464/154635876-2f047326-45f0-40d8-b2d4-6217c0a6c3a5.png)
**4-2 재고입력**
- 매장에 남아있는 재고의 수량을 수정하는 화면입니다.
- 실수량, 폐기량, 비고를 입력할 수 있으며 등록하기 버튼을 눌러 수정 내역을 저장합니다.


### 5. 메뉴관리
![menuList](https://user-images.githubusercontent.com/72987464/154635867-8ed539d6-5663-480e-8402-e20de9e16a1a.gif)
**5-1. 메뉴조회**
- 기업에 등록된 메뉴 목록을 보여주는 화면입니다.
- 필터 기능을 적용해 **전체보기, 카테고리별 보기, 메뉴명으로 검색하기**가 가능합니다.
- ‘제품보기’ 버튼을 클릭할 경우, 해당 메뉴에 등록된 판매제품 상세조회가 가능합니다.


![addMenu](https://user-images.githubusercontent.com/72987464/154635845-7cb05354-c18d-4049-81dc-b114903175c3.gif)
**5-2. 메뉴등록**
- 신메뉴를 등록할 수 있는 화면입니다.
- select box로 카테고리 선택 후, 메뉴코드, 메뉴명, 판매사이즈 여부를 선택 후 등록합니다.
- 다중 입력이 가능하도록 행 추가와 삭제 기능을 넣었습니다.


![Untitled (2)](https://user-images.githubusercontent.com/72987464/154635878-c16fb117-f025-4edb-8923-49701d4af117.png)
**5-3. 상세조회**
- 메뉴에 등록된 판매제품의 정보를 보여주는 화면입니다.
- **메뉴 이름, 사진, 판매사이즈, 기본정보, 재료정보, 메뉴변경내역**을 확인할 수 있습니다.
- ‘신규판매제품등록’ 버튼을 누를 경우 해당 메뉴에 대한 새로운 판매제품을 등록할 수 있습니다.


![addProduct](https://user-images.githubusercontent.com/72987464/154635860-64c600ed-8917-45af-bf8e-df35745bd3a4.gif)
**5-4. 신규판매제품등록**
- 메뉴에 대한 새로운 판매제품을 등록할 수 있는 화면입니다.
- **메뉴의 기본정보**와 **현재 ‘판매중’인 제품의 재료정보**가 자동으로 끌고와 집니다.
- 재료 수량 변경에 따른 **재료별 1인분가격, 원가, 마진율을 자동 계산**하도록 하였습니다.


![Untitled (3)](https://user-images.githubusercontent.com/72987464/154635881-3681338a-0038-47ec-8706-6f33a527fcff.png)
**5-5. 판매제품조회**
- 기업에 등록된 모든 판매제품 목록을 보여주는 화면입니다.
- 필터 기능을 적용해 **전체보기, 카테고리별 보기, 메뉴공개등급별 보기, 판매기간 구간 지정, 메뉴명으로 검색하기**가 가능합니다.


### 6. 주문관리
![030](https://user-images.githubusercontent.com/72987464/154635225-48b208bb-d79f-4956-aa39-e706e8b8e6af.png)
- 주문관리 화면입니다. 실시간으로 주문내역을 확인 할 수 있습니다.
- **진행중 주문**은 전체 주문 페이지에서 **초록색 배경**으로 표시되며 진행 중 주문 페이지에서도 실시간으로 확인 가능합니다.
- **주문번호**와 **메뉴명**으로 주문내역 검색이 가능합니다.


### 7. 매장관리
![034](https://user-images.githubusercontent.com/72987464/154635252-74bdfd30-9228-4ca4-885e-740271892135.png)
- 매장별 순이익과 총매출 비율을 확인할 수 있는 매장관리 화면입니다.
- 모든 매장에 대해 총 매출가격에 대한 순이익을 구하는 공식을 사용하여 **순이익이 높은 매장순**으로 정렬되어 보여집니다.
- 해당 매장 선택시 **해당 매장에 대한 총 매출액 비율 그래프**가 출력됩니다.


### 8. 공지
![040](https://user-images.githubusercontent.com/72987464/154635267-576fa8c0-1284-4f5f-af0b-727bec44197a.png)
- 공지사항 화면입니다. 각 부서 별로 공지를 등록하고 조회할 수 있습니다.
- 게시물은 등록 순으로 나열되며 제목, 내용, 부서명으로 특정 글을 검색 할 수 있습니다.
- 한 페이지에 10개의 글이 출력되며 그 이상의 게시물이 등록될 시 하단에 페이징바가 나타납니다.
