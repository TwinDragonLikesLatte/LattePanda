### 🦝🍰 프렌차이즈 카페 ERP 시스템, 라떼판다
- JSP 기반 웹 개발 프로젝트 **라떼판다**입니다.
- 프렌차이즈 카페 가맹, 직영점의 원활한 관리를 목적으로 합니다.
- 직급별로 볼 수 있는 화면을 달리하여 중요 정보를 선택적으로 제공하였습니다.
- 대시보드, 인사관리, 재고관리, 메뉴관리, 주문관리, 매장관리, 고객관리, 공지 기능을 제공합니다.
